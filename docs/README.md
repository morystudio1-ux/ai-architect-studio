# AI Architect Design Studio

AI destekli mimari tasarım platformu. 2D planlarınızdan lüks 3D modeller, gerçekçi renderlar ve profesyonel çıktılar oluşturun.

## 🚀 Hızlı Başlangıç

### Ana Site: https://morystudio1-ux.github.io/ai-architect-studio/
### Demo Sayfası: https://morystudio1-ux.github.io/ai-architect-studio/demo.html

## 🌟 Özellikler

- **Generative Design**: Yapay zeka ile tasarım varyasyonları oluşturun
- **3D Modelleme**: Gerçek zamanlı 3D model oluşturma
- **Ultra Render**: Sinema kalitesinde render çıktıları
- **Akıllı Analiz**: Performans, maliyet ve sürdürülebilirlik analizi
- **Çoklu Çıktı Formatları**: PNG/JPG, GLTF/OBJ, MP4, VR/AR, PDF

## 💻 Teknoloji

- HTML5 + CSS3 + Vanilla JavaScript
- Tailwind CSS + Font Awesome
- GitHub Pages deployment

## 🎮 Kullanım

1. Ana sayfadan "Tasarım Başlat" butonuna tıklayın
2. Planınızı sürükleyin/bırakın veya tıklayarak seçin
3. "AI ile Analiz Et" butonuna basın
4. İstediğiniz tasarım varyasyonunu seçin
5. "Render Başlat" ile kalite ayarlarını yapın
6. Çıktıları "Export" bölümünden indirin
